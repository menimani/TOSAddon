# Hawk-Eye
## *Description*
* Camconを魔改造したアドオン
* 設定フォルダ
    * Tree of Savior\addons\hawkeye

## *Commands*
* `/hawkeye` or `/hawk` : 
    * GUIの表示非表示を切り替える
* `/hawkeye z [number]` or `/hawk z [number]` : 
    * Z座標を指定した数値に変更する
    * X座標、Y座標についても同様
* `/hawkeye dismap` or `/hawk dismap` : 
    * コマンド入力したマップでHawk-Eyeが無効になる
* `/hawkeye reset` or `/hawk reset` : 
    * 視点を初期値(X45, Y38, Z236)に戻す
