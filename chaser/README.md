# Chaser
## *Description*
* 追従者を追う為のアドオン
    * 追従者出現マップへのトークンワープ
    * トークンワープのCDの把握
    * 追従者の討伐数の把握
    * チャットにて発見場所の共有
* 動作には別途「__m2util」が必要

## *Commands*
* `/chaser party <on/off>` : 
    * パーティチャットに報告する(on)かしないか(off)
    * デフォルトon

* `/chaser guild <on/off>` : 
    * ギルドチャットに報告する(on)かしないか(off)
    * デフォルトoff
