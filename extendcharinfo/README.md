# Extend Char info
## *Description*
* 下記右クリックメニューにキャラ情報確認を追加
    * チャット
    * フレンドリスト
    * ギルドメンバーリスト
* キャラ情報を確認してもいいねが勝手に飛ばない
* 自動マッチング中にささやきをチャット右クリックから送れる v1.0.3追加機能
* v1.3.0以降の動作には別途「__m2util」が必要

## *Commands*
* `/mem [チーム名]` : 
    * 対象のキャラ情報を確認出来る
    * 公式コマンド `/memberinfo [チーム名]` と違いいいねが勝手に飛ばない
